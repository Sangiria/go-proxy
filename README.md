# go-proxy

> ⚠️ Проект находится в активной разработке и является «сырым». Возможны баги, неполная функциональность и изменения без предупреждения.

Десктопное GUI-приложение для управления VLESS-прокси узлами на Linux. Позволяет добавлять серверы вручную или через подписку и запускать/останавливать проксирование одним кликом. Трафик маршрутизируется через [Xray-core](https://github.com/XTLS/Xray-core); SOCKS5-прокси поднимается на `127.0.0.1:20808`.

## Требования

- Linux x86_64
- Go ≥ 1.21
- Python ≥ 3.10
- `curl`, `unzip`, `rsync`

## Установка и сборка

### 1. Клонировать репозиторий

```bash
git clone https://github.com/Sangiria/go-proxy.git
cd go-proxy
```

### 2. Установить Python-зависимости

```bash
pip install PySide6 grpcio grpcio-tools protobuf pyinstaller
```

### 3. Скачать Xray

```bash
./install-xray.sh
chmod +x bin/xray   # скрипт может не выставить права — сделайте это вручную
```

### 4. Собрать приложение

```bash
make all
```

Результат окажется в `dist/ProxyApp/`.

### 5. Запустить

```bash
cd dist/ProxyApp
./run.sh
```

Скрипт запускает Go-бэкенд (gRPC на `:3333`) и GUI. При закрытии окна бэкенд завершается автоматически.

## Разработка (без сборки)

```bash
# Бэкенд
cd core && go run main.go

# UI (в отдельном терминале, из корня репозитория)
cd desktop && python main.py
```
